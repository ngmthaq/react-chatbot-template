# React Chatbot Template

Starter kit for a chatbot-style web app built with React 19, TypeScript, Vite, TanStack Router/Query, Material UI 7, Formik/Yup, and i18next. It ships with auth flows, theme and language toggles, and a sample chat UI you can wire to your own backend.

## Features

- React 19 + TypeScript with Vite for fast builds
- File-based routing via TanStack Router with generated `routeTree.gen.ts`
- Data layer ready through TanStack Query and Axios API helper
- Material UI 7 theme provider with light/dark toggle and CSS vars
- i18next with English and Vietnamese strings and language switcher
- Formik + Yup-based auth forms (login, register, forgot/reset password)
- Example chat layout with conversation list and message composer
- ESLint + Prettier + Husky + lint-staged for consistent code quality

## Tech Stack

- UI: Material UI, Emotion, styled-components
- Routing: TanStack Router + Devtools
- Data: TanStack Query, Axios
- Forms/validation: Formik, Yup
- I18n: i18next, react-i18next
- Date/time: Day.js

## Project Structure

```
src/
├── assets/                # Static assets (CSS, images)
├── core/                  # Reusable core logic
│   ├── constants/         # API endpoints, keys, storage keys
│   ├── hooks/             # Data/auth hooks
│   ├── providers/         # App-level providers (theme, i18n, query)
│   ├── schemas/           # Yup schemas
│   ├── types/             # Type definitions
│   └── utils/             # API client, storage, helpers
├── routes/                # File-based routes for TanStack Router
│   ├── _guest/            # Public layouts and pages (login, register, password)
│   └── _auth/             # Authenticated layouts and pages (chat, profile)
└── main.tsx               # Router bootstrap
```

## Getting Started

Prerequisites: Node.js 18+ and npm/pnpm/yarn.

1. Install dependencies

```bash
yarn install
```

2. Run the dev server

```bash
yarn dev
```

4. Open http://localhost:5173 and explore the guest (login/register) and auth (chat/profile) routes.

## Available Scripts

- `yarn dev` — start dev server with HMR
- `yarn build` — type-check then build for production
- `yarn preview` — preview the production build locally
- `yarn lint` — run ESLint
- `yarn lint:fix` — auto-fix lint issues
- `yarn format` — format with Prettier

## Key Implementation Notes

- Routing: Routes live in `src/routes`; `routeTree.gen.ts` is generated by TanStack Router.
- Providers: `AppLocalizationProvider`, `AppQueryProvider`, and `AppThemeProvider` wrap the app in [`routes/__root.tsx`](src/routes/__root.tsx).
- Theming: Uses MUI color schemes with a header toggle in the guest layout.
- I18n: Translation files live in `src/core/providers/AppLocalizationProvider/lang/` (`en.json`, `vi.json`).
- API client: `src/core/utils/api.ts` initializes Axios with `window.__ENV__.VITE_PUBLIC_API_BASE_URL`.

## Extending the Template

- Wire the chat UI to your API by replacing the placeholder state in `src/routes/_auth/chat/index.lazy.tsx` with TanStack Query hooks.
- Swap the mock auth hooks in `src/core/hooks/` with real endpoints and persist tokens using `src/core/utils/storage.ts`.
- Add new routes under `src/routes`, then regenerate the route tree (`yarn dev` handles this automatically).
